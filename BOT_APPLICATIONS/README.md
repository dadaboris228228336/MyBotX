# BOT APPLICATIONS - Приложения для работы бота

Эта папка содержит все необходимые приложения для автономной работы MyBotX.

### 🐍 Python 3.10.11 ✅ СКАЧАН
- **Файл:** `python-3.10.11-amd64.exe`
- **Размер:** 29 MB
- **Статус:** ✅ Готов к установке
- **Описание:** Интерпретатор Python с tkinter и pip

### 📱 BlueStacks 5.22.130.1019 ✅ СКАЧАН
- **Файл:** `BlueStacksInstaller_5.22.130.1019_amd64_native.exe`
