# MyBotX — Версии зависимостей

## Среда выполнения

| Компонент   | Версия          | Примечание                                      |
|-------------|-----------------|--------------------------------------------------|
| Python      | 3.10.11         | Зафиксирована. Код использует синтаксис 3.10+   |
