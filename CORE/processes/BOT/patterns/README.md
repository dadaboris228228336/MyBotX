# 🖼️ Папка паттернов (шаблонов кнопок)

Сюда кладём PNG скриншоты кнопок из Clash of Clans.

## Как добавить паттерн:

1. Запустите игру в BlueStacks
2. Сделайте скриншот нужной кнопки (вырежьте только кнопку, без лишнего фона)
3. Сохраните в эту папку с нужным именем

## Список паттернов:

| Файл                        | Описание                    |
|-----------------------------|-----------------------------|
| btn_attack.png              | Кнопка "Атака"              |
| btn_close.png               | Кнопка закрытия окна (X)    |
| gold_mine_full.png          | Золотая шахта (полная)      |
| elixir_collector_full.png   | Сборщик эликсира (полный)   |
| dark_elixir_full.png        | Тёмный эликсир (полный)     |
| btn_find_match.png          | Кнопка "Найти матч"         |
| btn_next.png                | Кнопка "Далее"              |

## Советы:

- Вырезайте кнопку точно, без лишних пикселей
- Размер паттерна: 50x50 до 200x200 пикселей
- Порог совпадения по умолчанию: 0.8 (80%)
- Если паттерн не находится — попробуйте порог 0.7
